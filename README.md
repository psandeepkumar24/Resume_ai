# AI Resume Screening System

An AI-powered resume screening application that compares resumes with job descriptions using Natural Language Processing (NLP).

## Features
- Resume PDF Upload
- Job Description Analysis
- TF-IDF Vectorization
- Cosine Similarity Matching
- Candidate Recommendation

## Technologies Used
- Python
- Streamlit
- Scikit-Learn
- PyPDF2

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
